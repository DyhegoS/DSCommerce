# Sobre o projeto

DSCommerce é uma aplicação de e-commerce desenvolvida utilizando como backend Java + SpringBoot e no frontend Angular.

Consiste em uma aplicação que realiza a criação, leitura, atualização e deleção (CRUD) de produtos e suas categorias e também pedidos. Ao realizar um pedido é gerado um pagamento com status de "Aguardando pagamento", possui também login de usuários, autorização de acessos. O sistema conta com testes unitários e de integração das funções citadas.

A parte frontend ainda está em desenvolvimento.

# ▶️ Como executar o projeto
**Pré-requisitos: Java 17**

## Clonar o repositório
```bash
git clone https://github.com/DyhegoS/DSCommerce.git
```

## Entre na pasta do projeto clonado
```bash
cd DSCommerce
```

## execute o projeto
```bash
./mvnw spring-boot:run
```
